Inputs de Botones

Los inputs de tipo botón permiten ejecutar acciones dentro de un formulario web.

Nota: Estos botones se usan como práctica para observar su funcionamiento.

1. Input type="button"

Botón genérico que no realiza ninguna acción por sí solo.

Este ejemplo muestra cómo un botón puede ejecutar una acción, en este caso generar un nombre aleatorio y mostrarlo en el campo de texto.




2. Input type="submit"

Envía el formulario y permite validar los datos ingresados.




3. Input type="reset"

Restablece todos los campos del formulario a sus valores iniciales.




4. Input type="image"

Funciona como botón de envío utilizando una imagen personalizada.
