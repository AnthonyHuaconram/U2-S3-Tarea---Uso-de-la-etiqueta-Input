Inputs Especiales

Los inputs especiales son aquellos que cumplen funciones técnicas específicas dentro de formularios web.

Nota: Los botones son solo para práctica del funcionamiento de los inputs.

1. Input type="number"

Permite ingresar únicamente valores numéricos en un campo controlado.



2. Input type="hidden"

Campo oculto que almacena información sin mostrarse al usuario.

Después de enviar el formulario, este dato (como un ID o código interno) se envía al servidor sin que el usuario lo vea ni lo pueda modificar.
