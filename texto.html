Inputs de Texto

Los inputs de tipo texto permiten ingresar información escrita en formularios web de manera sencilla.

Nota: Los botones son solo para práctica del funcionamiento de los inputs.

1. Input type="text"

Permite ingresar texto libre como nombres, direcciones o información general.



2. Input type="password"

Oculta los caracteres ingresados para proteger la información sensible del usuario.



3. Input type="email"

Valida automáticamente que el formato ingresado corresponda a un correo electrónico válido.





4. Input type="tel"

Permite ingresar números telefónicos, pero no valida automáticamente su formato, ya que estos varían según el país o región.



5. Input type="url"

Permite ingresar direcciones web y valida que tengan un formato correcto.



6. Input type="search"

Permite ingresar términos de búsqueda; funciona como un campo de texto, pero puede tener un estilo visual diferente.
