Bienvenido a mi sitio web

Este sitio web tiene como objetivo mostrar los diferentes tipos de inputs disponibles en HTML, utilizados para la creación de formularios interactivos.

Cada sección presenta una categoría de inputs con su respectiva explicación y ejemplo práctico.

Categorías disponibles:

  • Texto: Inputs para ingresar información textual.
  • Fechas: Inputs relacionados con fecha y hora.
  • Selección: Inputs para elegir opciones.
  • Especiales: Inputs con funciones específicas.
  • Botones: Inputs para acciones dentro del formulario.
