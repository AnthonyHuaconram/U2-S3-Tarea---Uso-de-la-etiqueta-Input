Inputs de Selección

Los inputs de selección permiten elegir opciones, colores o archivos dentro de un formulario web.

Nota: Los botones son solo para práctica del funcionamiento de los inputs.

1. Input type="checkbox"

Permite seleccionar una o varias opciones al mismo tiempo.

Deportes Música Videojuegos Lectura

2. Input type="radio"

Permite seleccionar solo una opción entre varias disponibles.

Masculino Femenino Otro

3. Input type="color"

Permite seleccionar un color desde una paleta visual.



4. Input type="range"

Permite seleccionar un valor numérico dentro de un rango definido usando una barra deslizante.

5. Input type="file"

Permite subir archivos desde el dispositivo del usuario.
