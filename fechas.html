Inputs de Fecha y Tiempo

Los inputs de fecha permiten al usuario seleccionar fechas y horarios de manera sencilla dentro de formularios web.

Nota: Los botones son solo para práctica del funcionamiento de los inputs.

1. Input type="date"

Permite seleccionar una fecha completa como día, mes y año.



2. Input type="time"

Permite seleccionar una hora específica del día.



3. Input type="datetime-local"

Permite seleccionar fecha y hora en un solo campo sin zona horaria.



4. Input type="month"

Permite seleccionar únicamente el mes y el año.



5. Input type="week"

Permite seleccionar una semana específica del año.
